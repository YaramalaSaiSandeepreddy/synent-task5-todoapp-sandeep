NexTask — Futuristic To-Do App 🚀

A cyberpunk-inspired, fully responsive Task Management Web Application built with pure HTML, CSS, and Vanilla JavaScript. Features glassmorphism UI, smooth animations, drag-and-drop reordering, and full localStorage persistence.


🌐 Live Demo


Open index.html directly in any browser — no server needed.




📸 Features

✅ Core Features


Add tasks using button click or pressing Enter
Edit tasks inline with instant save
Delete tasks with smooth slide-out animation
Mark tasks as Complete / Incomplete with animated checkbox
All tasks saved to localStorage — data persists after refresh


🎯 Priority System


🔴 High — Red glow indicator
🟠 Medium — Orange glow indicator
🔵 Low — Blue glow indicator


📅 Due Dates


Set due dates on each task
Overdue tasks automatically show a ⚠️ red warning


🔍 Search & Filter


Real-time search as you type (case-insensitive)
Filter by: All / Active / Completed


📊 Task Statistics


Live count of Total, Pending, and Completed tasks


🌙 Dark / Light Mode


Toggle between dark and light themes
Theme preference saved in localStorage


🖱️ Drag & Drop


Reorder tasks by dragging and dropping
New order saved to localStorage automatically


🔔 Toast Notifications


Animated toast messages for every action:

✅ Task Added
🗑️ Task Deleted
✏️ Task Updated
🧹 Completed Tasks Cleared
🌙 Theme Changed





🧹 Clear Completed


One-click button to remove all completed tasks
